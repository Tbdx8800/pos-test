Cómo Usar el Sistema

Bienvenido a RealPhone POS

Este sistema te permite realizar ventas, gestionar tu inventario y emitir tickets fácilmente.

  • Ventas: Busca un producto o presiona la tecla F10. Presiona Enter para agregarlo al ticket. Para cobrar la venta presiona F12.
  • Inventario: Los administradores pueden presionar F8 o entrar a "Configuración" para gestionar, agregar o eliminar productos.
  • Cortes: Usa F4 para realizar corte parcial o completo de caja al finalizar el turno.
  • Gestor de Tickets: Haz clic en el ícono de celular arriba a la izquierda para cambiar a la pantalla de servicio técnico (Reparaciones, Apartados, Encargos).

Si tienes problemas, consulta con tu administrador de tienda.
